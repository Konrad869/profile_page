Opis mojego portfolio

Advertising_agency

Advertising_agency to strona prezentująca witrynę internetową agencji reklamowej, stworzona z wykorzystaniem technologii HTML 5, CSS3, JavaScript, Sass. Strona utrzymuje standardy RWD i SEO.

coffeehouse_page

Coffeehouse_page to strona prezentująca witrynę kawiarni. Strona została wykonana w technologii HTML i CSS, jest w pełni responsywna i zoptymalizowana pod kątem SEO.

Pizzeria_page_

Strona Pizzeria_page_ to strona reklamująca pizzerię, technologie użyte do stworzenia strony to HTML, CSS, Sass, JavaScript w oparciu o metydologie ITCSS i BEM, strona jest w pełni responsywna i zoptymalizowana pod kątem SEO.

Weather_application_IMGW

Aplikacja Weather_application_IMGW to aplikacja odczytująca dane z IMGW przy wykorzystaniu API Instytutu Metyrologii.

Game_database

Aplikacja Game_database to aplikacja, która dodaje gry wraz z nazwą, oceną i opisem do bazy gier oraz umożliwia ich usunięcie z bazy danych.

Angular_Design

Aplikacja Angular_Design to aplikacja Angular, która dodaje użytkowników do bazy danych. Aplikacja jest jeszcze nieukączona, ale pracuję nad jego ukączeniem. Aplikacja zbudowana jest przy użyciu technologii Angular v16, Typescript, Rxjs, SPA, Angular material . Aby ją uruchomić, pobierz pliki ZIP lub użyj polecenia git colne plus adres i otwórz w IDE. W katalogu src gdzie musisz zainstalować ng_modules za pomocą npm install, a następnie wydaj polecenie ng s -o lub ng serve aby ją uruchomić.
